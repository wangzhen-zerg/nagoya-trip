# 名古屋旅行

名古屋旅行规划助手 - 一个简洁实用的日本名古屋旅行规划工具。

## 功能特点

### 🗺️ 地图标注
- 使用 Leaflet + OpenStreetMap 显示交互式地图
- 支持添加、编辑、删除地点标记
- 拖拽排序调整游览顺序
- 自动调整视图显示所有标记

### 🔍 智能搜索
- 本地预设 25+ 名古屋热门景点
- 支持在线搜索（Photon API / Nominatim）
- 中日文双语搜索支持

### 🍜 逛吃推荐
- 预置名古屋特色美食推荐
- 支持添加自定义美食
- 标记需预约店铺和休息日
- 记录人均价格和备注

### 🚗 路线规划
- 支持驾车/步行/公交三种出行方式
- 自动计算预估费用和时间
- 高速费和油费明细
- 费用换算（人民币显示）

### 📝 旅行备忘录
- 记录重要旅行信息
- 自动保存到本地

### 💾 数据持久化
- 所有数据自动保存到浏览器本地存储
- 刷新页面不丢失数据

## 技术栈

- **地图**: Leaflet + OpenStreetMap
- **搜索API**: Photon (komoot) / Nominatim
- **路由服务**: OSRM (Open Source Routing Machine)
- **代码**: 原生 JavaScript (ES6+)
- **样式**: CSS3（CSS变量 + Flexbox）

## 使用方法

### 方式一：直接打开
双击 `index.html` 在浏览器中打开即可使用。

### 方式二：本地服务器
```bash
# 使用 Python 3 启动本地服务器
python3 -m http.server 8080

# 或使用 npm
npm run dev

# 然后在浏览器中访问 http://localhost:8080
```
